Optional Parameters

Plugin parameters
Parameter Type Default Description
useLabels boolean true This option lets you specify whether to use <label> elements or <div> to emulate your placeholder. Depricated.
phClass string "placeholder" The class to attach to the placeholder elements so you can style them.
idPrefix string "ph-" The id prefix used on the placeholder elements. May be removed down the line in favor of data-*.
css object color : '#aaa' An object containing any standard jQuery css object key/value pairs (color being the only property not stylable by your own stylesheets).

Known Issues

Because of the way I emulate native htm5 placeholder behavior, the color of the plceholder text is placed inline on the elements and can not be styled by your stylesheet. Everything else, however, can be. But you can customize the color when you setup the .placeHolder() object.
